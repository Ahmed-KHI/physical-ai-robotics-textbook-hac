## üì¶ Creating a Package

```bash
cd ~/ros2_ws/src
ros2 pkg create --build-type ament_python my_robot_pkg \
  --dependencies rclpy std_msgs sensor_msgs
```

## üöÄ Launch Files

Launch multiple nodes with one command:

```python
from launch import LaunchDescription
from launch_ros.actions import Node

def generate_launch_description():
    return LaunchDescription([
        Node(
            package='my_robot_pkg',
            executable='sensor_node',
            name='sensor',
            parameters=[{'frequency': 10}]
        ),
        Node(
            package='my_robot_pkg',
            executable='controller_node',
            name='controller'
        ),
    ])
```

Run with:
```bash
ros2 launch my_robot_pkg robot.launch.py
```

## ‚öôÔ∏è Parameters

Make nodes configurable:

```python
self.declare_parameter('max_speed', 1.0)
max_speed = self.get_parameter('max_speed').value
```

Set from command line:
```bash
ros2 run my_pkg my_node --ros-args -p max_speed:=2.0
```
